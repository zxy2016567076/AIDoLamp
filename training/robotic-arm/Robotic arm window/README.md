# 四轴机械臂台灯跟踪系统

这个项目实现了一个基于计算机视觉的智能台灯系统，利用四轴机械臂和摄像头，能够自动跟踪并照亮桌面上的书本或其他物体。

## 系统架构

系统由三个主要模块组成：

1. **目标检测模块 (yolo.py)**：使用YOLOv8实时检测桌面上的物体，并将图像坐标转换为机械臂坐标系坐标
2. **机械臂控制模块 (robetpose.py)**：实现机械臂的逆运动学和正运动学计算，确保台灯以合适的角度照射目标
3. **跟踪控制模块 (tracking_control.py)**：整合上述两个模块，实现物体检测和机械臂协调控制的功能

## 工作原理

### 坐标系转换

系统解决的核心问题是将摄像头图像中的2D坐标转换为3D机械臂坐标系：

1. 摄像头捕获图像，YOLOv8检测目标位置（像素坐标）
2. 系统估计目标与摄像头的距离（通过已知物体尺寸或测距模块）
3. 使用针孔相机模型将2D像素坐标+距离转换为3D世界坐标
4. 考虑机械臂末端执行器当前位置与姿态，转换为机械臂基座坐标系
5. 使用逆运动学计算各关节角度，使台灯末端朝向目标

### 平滑跟踪

为避免机械臂频繁抖动，系统采用了多种平滑策略：

- 移动阈值：只有当目标移动距离超过设定阈值时才启动机械臂
- 历史轨迹平滑：使用加权平均处理历史位置数据
- 渐进式移动：使用平滑因子在当前位置和目标位置之间插值

## 使用方法

### 安装依赖

```bash
pip install opencv-python numpy torch ultralytics matplotlib
```

### 运行系统

```bash
python tracking_control.py
```

### 功能选项

运行后系统会显示以下选项：

1. **开始跟踪**：开始自动跟踪桌面上的物体
2. **校准系统**：使用已知尺寸的物体（如A4纸）校准距离估计
3. **测试模式**：只进行物体检测和坐标转换，不控制机械臂（调试用）
4. **退出**：退出程序

### 校准说明

首次使用前建议进行校准：
1. 选择"校准系统"
2. 将A4纸放在摄像头前约40cm处
3. 当检测到A4纸时，按C键校准
4. 校准完成后，系统将更准确地估计目标距离

## 系统配置

可以在`tracking_control.py`中修改以下参数：

- `camera_index`：摄像头索引，默认为0
- `model_path`：YOLO模型路径，默认使用yolov8n.pt
- `use_distance_sensor`：是否使用测距传感器，默认为False
- `tracking_interval`：跟踪更新间隔，默认0.2秒
- `visualization`：是否显示可视化界面，默认为True

## 可扩展功能

### 添加测距模块

如果希望提高距离估计精度，可以添加实际的测距传感器（如超声波或ToF传感器）：

1. 修改`yolo.py`中的`read_distance_module()`函数，实现与实际传感器的通信
2. 运行系统时将`use_distance_sensor`参数设为True

### 添加机械臂控制

当前系统计算出关节角度后，需要添加与实际机械臂通信的代码：

1. 修改`tracking_control.py`中的`_tracking_loop()`函数
2. 在注释"在这里添加实际控制机械臂的代码"处，添加与舵机控制器通信的代码

## 故障排除

1. **检测不准确**：尝试校准系统或使用更好的YOLO模型
2. **坐标转换不准确**：检查摄像头安装位置是否与代码假设一致
3. **机械臂移动不平滑**：调整`min_movement_threshold`和`smoothing_factor`参数
4. **摄像头无法打开**：检查`camera_index`设置是否正确

## 文件说明

- `yolo.py`: 目标检测与坐标转换模块
- `robetpose.py`: 机械臂逆/正运动学计算模块
- `tracking_control.py`: 系统主控制模块
